Como o carrinho foi montado?

O carrinho da nossa equipe foi construído com:

  • um chassi feito de resina preta,
  • uma esfera,
  • uma placa Arduino Uno,
  • um sensor ultrassônico,
  • um sensor de linha,
  • uma ponte H,
  • dois motores,
  • uma bateria de 9V,
  • e quatro pilhas.
